Links for August 24th (del.icio.us)

Aug 24th, 2009

These are semanticdreamer’s del.icio.us bookmarks for August 24th from 07:13 to 10:38:

  • Sindice - The semantic web index - is based on DERI’s unique cluster technology which indexes and operates over terascale semantic data sets (trillions of statements) while also providing very high query throughputs per cluster size. Over 10 billion pieces of reusable information can already be found across 100 million web pages which embed RDF and Microformats. indice is created at DERI, the world’s largest institute for Semantic Web research.
  • Welcome to OKKAM - Enabling the Web of Entities - A web-scale open service called Entity Name System (ENS) for supporting the systematic reuse of identifiers for "things" - project co-funded by the European Commission.
  • Acre - Freebase’s Application Hosting Platform - an application hosting platform designed to minimize the effort required by the Freebase community to build web applications that make use of the data contained by Freebase and create mashups with web services provided by others.
  • Personas - is a component of the Metropath(ologies) exhibit, currently on display at the MIT Museum by the Sociable Media Group from the MIT Media Lab. It uses sophisticated natural language processing and the Internet to create a data portrait of one's aggregated online identity. In short, Personas shows you how the Internet sees you.
