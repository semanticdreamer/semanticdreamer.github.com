About

Semantic Dreamer is a digital lifestyle journal written by Matthias Geisler.

Selection of main topics: open Source software (OSS), open web standards, the semantic web, internet technology in general, getting things done (GTD).

The website is powered by Octopress (a blogging framework for Jekyll, the blog aware static site generator powering GitHub Pages) and hosted on GitHub.

A little something about me.

I’m Matthias Geisler, a Technical Architect and Project Manager living and working in/ near Donauwoerth, Germany.
